# Raster Docs

This directory contains the Raster documentation site built with Astro
Starlight.

## Commands

- `pnpm --dir docs dev`: start the local docs server.
- `pnpm --dir docs build`: build the static docs site.
- `pnpm --dir docs preview`: preview the built docs site.

Root-level aliases are also available: `pnpm run docs:dev`,
`pnpm run docs:build`, and `pnpm run docs:preview`.
